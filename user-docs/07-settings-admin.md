# 7. Настройки (для администраторов)

> Раздел «Настройки» доступен только пользователям с административными правами. Если вы не видите этот раздел в меню — обратитесь к администратору.

---

## Как открыть настройки

В левом меню нажмите **Настройки**. Откроется страница с четырьмя карточками.

---

## Пользователи

Здесь создаются и управляются аккаунты сотрудников для входа в систему.

### Как добавить пользователя

1. Откройте **Настройки → Пользователи**.
2. Нажмите **Добавить пользователя**.
3. Заполните форму:
   - **Полное имя** — как отображается в системе.
   - **Email** — будет использоваться для входа.
   - **Пароль** — временный пароль (сотрудник сможет сменить его после входа).
   - **Роль** — выберите одну или несколько ролей из списка.
4. Нажмите **Сохранить**.

### Управление пользователями

В таблице пользователей можно:
- **Редактировать** — изменить имя, email, роль.
- **Активировать / Деактивировать** — включить или отключить доступ в систему.

Деактивированный пользователь не может войти в систему, но его данные сохраняются.

---

## Роли и привилегии

Здесь создаются и настраиваются роли — наборы прав, которые назначаются пользователям.

### Как создать новую роль

1. Откройте **Настройки → Роли и привилегии**.
2. Нажмите **Добавить роль**.
3. Заполните:
   - **Название роли** — понятное имя, например «Менеджер набора».
   - **Код роли** — короткий уникальный идентификатор, только латинские буквы и цифры.
   - **Категория** — выберите из списка (Образование, Финансы, Управление и т. д.).
   - **Описание** — опционально, поясните назначение роли.
4. Нажмите **Сохранить**.

### Как настроить права роли

После создания роли выберите её в списке. Откроется таблица модулей. Для каждого модуля можно включить:
- **Просмотр** — видит данные модуля.
- **Создание** — может создавать новые записи.
- **Редактирование** — может изменять существующие записи.
- **Удаление** — может удалять записи.

> Системные роли (отмечены значком «Системная») изменять нельзя.

---

## Справочник городов

Список городов, который используется при заполнении адресов лидов, сотрудников и других записей.

### Как добавить город

1. Откройте **Настройки → Справочник городов**.
2. Нажмите **Добавить**.
3. Введите страну и название города.
4. Нажмите **Сохранить**.

---

## Справочник должностей

Список должностей, доступных при оформлении сотрудников.

### Как добавить должность

1. Откройте **Настройки → Справочник должностей**.
2. Нажмите **Добавить**.
3. Введите название должности.
4. Нажмите **Сохранить**.

Должности делятся на типы: преподавательские, управленческие, вспомогательные.

---

## Управление отделами

Структура отделов управляется в модуле **Персонал**, а не в Настройках. Подробнее — в разделе [06-staff-management.md](06-staff-management.md).

---

## Рекомендации для администратора

- Регулярно проверяйте список пользователей. Деактивируйте аккаунты уволенных сотрудников сразу.
- Используйте минимально необходимые права: не давайте роль с полным доступом там, где достаточно ограниченной.
- Системные роли не трогайте без необходимости.
- При создании новой роли обязательно протестируйте её, войдя в систему под тестовым аккаунтом с этой ролью.
