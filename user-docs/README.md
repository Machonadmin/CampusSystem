## Быстрый старт

1. Войдите в систему — введите email и пароль на странице входа.
2. На главной странице нажмите на нужный модуль.
3. Для работы с лидами — откройте **Образование → Набор**.
4. Для задач — откройте модуль **Задачи**.

## Обратная связь

Если в документации что-то непонятно или устарело — обратитесь к администратору системы.
