# מדריך למנהל מערכת / מנהל קמפוס

מדריך זה מיועד ל**מנהל הקמפוס ולמנהל־על**. הוא מכסה את מה שייחודי לניהול:
הגדרות, משתמשים והרשאות, מבנה ארגוני, תבניות תהליך, ומשימות.

לפני שממשיכים — קראו את [01-התחלה](01-התחלה.md) ואת
[02-תפקידים-והרשאות](02-תפקידים-והרשאות.md).

---

## גישה מלאה

ל**מנהל־על** יש גישה לכל המודולים והפעולות ללא הגבלה. **מנהל קמפוס** נוחת על
המסך הראשי עם כל המודולים הזמינים לו.

## הגדרות — תפקידים והרשאות

במסך **הגדרות** מנהלים את התפקידים וההרשאות:

- **תפקידים**: יצירה ועריכה של תפקידים, עם שמות בשלוש שפות. כל תפקיד מקבל חבילת
  הרשאות למודולים ולפעולות.
- **קטלוג הרשאות**: הוספת הרשאות (שם + קוד) ושיוכן לתפקידים.
- **מודולים זמינים**: קביעה לאילו מודולים לתפקיד יש גישה.

## משתמשים והרשאות אישיות

במסך **הגדרות ← משתמשים**:

- יצירת כניסה (משתמש) לעובד קיים (אפשר גם מתוך מודול כוח אדם).
- **הרשאות אישיות**: מעבר לתפקיד, אפשר להוסיף או לשלול הרשאה נקודתית לאדם מסוים.
  הרשאה אישית גוברת על הרשאת התפקיד.

## מבנה ארגוני

את העץ הארגוני (מחלקות, ראשי מחלקות, תת-מחלקות) מנהלים במודול **כוח אדם ←
מבנה** (ראו [מדריך משאבי אנוש](role-משאבי-אנוש.md)). ההרשאות בחלק מהמודולים
מתחשבות במחלקה — ראש מחלקה רואה את תת-המחלקות שלו.

## תבניות תהליך (Workflow)

תהליכי הקבלה בנויים משלבים, חותמים ומעברים. במסך תבניות התהליך אפשר לנהל את
השלבים (יצירה/עריכה/מחיקה), את החותמים, ואת המעברים ביניהם. כך נקבע איך זורם
תהליך הקבלה של המועמדות.

## משימות

מודול **משימות** (`/dashboard/tasks`) משמש את כולם, ושימושי במיוחד לניהול:

- **תצוגות** (לשוניות): שהוקצו לי / שיצרתי / מחלקה / במעקב.
- **מסננים**: סטטוס ודחיפות.
- **משימה חדשה**: כותרת, תיאור, חד-פעמית או חוזרת, הקצאה (אליי / אדם / מחלקה),
  דחיפות, צופים, ותאריך יעד. משימה חוזרת כוללת כלל חזרה (יומי/שבועי/חודשי/שנתי)
  עם תצוגה מקדימה של המופע הבא.
- **פעולות מרוכזות**: «מצב בחירה» ואז **סימון כבוצע** או **הקצאה** לאדם למספר
  משימות בבת אחת.
- **בכרטיס המשימה**: מעברי סטטוס (תפיסה, התחלה, דחייה, שליחה לבדיקה, אישור,
  פתיחה מחדש) לפי היותכם יוצר/מבצע, תגובות, וצופים. אפשר לבטל סדרה שלמה (מהיום
  והלאה או הכול) עם תצוגה מקדימה של מה יימחק ומה יישמר.

## דוחות

מודול **דוחות** מרכז לוחות לכל התחומים (משפך קבלה, תלמידות, כספים, מעונות,
הזנה, תחזוקה, מרפאה, ייעוץ, מסמכים, תורמים, אבטחה). מסנן התקופה משפיע על
כרטיסי הכספים והתורמים בלבד. הדוחות הם לצפייה (אין ייצוא CSV במסך הדוחות; ייצוא
זמין ברשימות עצמן — לידים, אבטחה, רפואה וכו').

## לפני עלייה לאוויר

ראו את נוהל **מוכנוּת להשקה** במסמכי התפעול: `docs/ops/launch-readiness.md`
(אימות מיגרציות, משתני סביבה, Sentry, גיבויים).

## טיפים

- אל תשתפו את משתמש מנהל־העל; העניקו לכל אחד תפקיד מתאים.
- חיפוש מהיר: `Ctrl/Cmd + K`.
