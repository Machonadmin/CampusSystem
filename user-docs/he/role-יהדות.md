# מדריך לאחראי בירור יהדות

מדריך זה מיועד ל**אחראי בירור יהדות**. הוא מכסה את מודול **בירור יהדות** ואת
הקשר שלו לוועדת הקבלה.

לפני שממשיכים — קראו את [01-התחלה](01-התחלה.md) ואת
[02-תפקידים-והרשאות](02-תפקידים-והרשאות.md).

---

## איפה עובדים

אחרי הכניסה אתם נוחתים על מודול **בירור יהדות** (`/dashboard/jewishness`).
הגישה למודול היא הרשאה אחת («גישה») — מי שיש לו גישה רואה את הכול.

## הרשימה

- **סינון לפי סטטוס** בלשוניות עם מונים: הכל / ממתין / מאושר / נדחה / דורש בדיקה.
- **חיפוש** לפי שם או אימייל.
- כל שורה מציגה שם, שם עברי, תגית סטטוס, תגית **«בקבלה»** (כשהמועמדת בשלב קבלה
  פעיל), מספר מסמכים, וכפתור **«פרטים»**.

## חלון הפרטים — מה עושים

לחיצה על «פרטים» פותחת את כרטיס הבירור:

- **צפייה**: פרטים אישיים (אימייל, תאריך לידה, אזרחות), הסטטוס הנוכחי (מי החליט,
  מתי, הערות), **היסטוריית סטטוסים** מלאה, ומסמכים.
- **קביעת סטטוס**: הערה (רשות) ואחד מארבעה: **ממתין / מאושר / נדחה / דורש בדיקה**.
- **מסמכים**: פתיחת מסמך קיים (נפתח בלשונית חדשה) והעלאת מסמך חדש (כותרת + קובץ).
- **קישור לקבלה**: כאשר המועמדת נמצאת בשלב קבלה פעיל, מופיע אזור החלטה — בוחרים
  את התוצאה הסופית (אושר / נדחה), מוסיפים הערה, ו**חותמים** (חתימה מצוירת או
  מוקלדת). החתימה משלימה את שלב הקבלה ומזרימה את החלטת היהדות אל תהליך הקבלה
  הרב-שלבי.

> אישור היהדות הוא **עצמאי** — הוא אינו חוסם אישורים אחרים בוועדה, וגם אינם
> חוסמים אותו.

## טיפים

- אם מועמדת אינה מופיעה — ודאו שהיא הומרה מליד למועמדת בצד הגיוס.
- חיפוש מהיר: `Ctrl/Cmd + K`.
