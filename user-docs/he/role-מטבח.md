# מדריך למטבח / הזנה

מדריך זה מיועד ל**ראש מטבח** ולצוות ההזנה. הוא מכסה את מודול **הזנה**: תוכניות
הזנה, רישום תלמידות, ופרופיל תזונתי.

לפני שממשיכים — קראו את [01-התחלה](01-התחלה.md) ואת
[02-תפקידים-והרשאות](02-תפקידים-והרשאות.md).

---

## מבנה המודול

מודול **הזנה** (`/dashboard/food`) מנהל **תוכניות הזנה** ורישום תלמידות אליהן
(לא תפריט יומי או הזמנות בודדות).

## רשימת תוכניות

כרטיס לכל תוכנית: שם, קוד, תווית «לא פעילה» בעת הצורך, סוגי ארוחות (בוקר/צהריים/
ערב), מספר רשומות פעילות, ומחיר.

- **«+ הוספת תוכנית»** (לפי הרשאת ניהול): שם (חובה), קוד, מחיר, תווית תקופה,
  וסימון אילו ארוחות כלולות.
- לחיצה על כרטיס פותחת את פרטי התוכנית.

## פרטי תוכנית

- **רישום תלמידה** (לפי הרשאת ניהול): חיפוש תלמידה (מוצגת התוכנית הנוכחית שלה),
  תאריך התחלה (חובה) ותאריך סיום.
- **טבלת רשומות**: תלמידה, מתאריך, עד תאריך, סטטוס (פעיל / הסתיים). לכל שורה
  פעילה: **עריכת מגבלות תזונה** ו**סיום רישום**.
- **עורך תזונה**: מגבלות, אלרגיות והערות לתלמידה.

## טיפים

- רישום מסתיים דרך «סיום רישום» כדי לשמור היסטוריה.
- חיפוש מהיר של תלמידה: `Ctrl/Cmd + K`.
