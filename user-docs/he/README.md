# מדריך למשתמש — CampusSystem (מכון חמש)

מערכת CampusSystem היא מערכת ניהול הקמפוס: גיוס ומיון מועמדות, ניהול לימודים,
צוות, כספים, מעונות, בירור יהדות ועוד. כל הנתונים במקום אחד, נגישים מכל מכשיר
דרך הדפדפן.

המדריך בנוי כך שכל בעל תפקיד יקרא קודם את שני הפרקים הכלליים, ואז את המדריך
המתאים לתפקיד שלו.
